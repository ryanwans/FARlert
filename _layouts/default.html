Why choose FARlert?

FARlert simplifies the process of creating and displaying dialogs for your website, while also allowing for maximum customization. The alerts mimic that of Windows 10 dialogs, to give a professional and clean look to them.

Documentation

Creating an Alert

var options = {...};
var SecurityAlert = new FAR.popup(options);

Hoisting an Alert

// Hoist (show) Alert
SecurityAlert.hoist();
// Retract (hide) Alert
SecurityAlert.retract();

Valid Alert Options

Name Description Default
borderFlash Flash border upon hoisting false
moveable Move around the dialog box true
ding Play sound upon hoisting false
title The title at top of the alert box 'Alert'
html The main inner-contents of the alert null
jQuery Choose to disable usage of jQuery automatic
escapeKey Use the escape key to exit alerts true
newWindow Open the alert in a seperate window false
buttons The name and function of buttons on alert SEE BELOW

Defining Alert Buttons

When defining buttons in the Options object, it should be an Array of Object, containing the 'name' and 'func' of that button:

var options = {
 . . .
 buttons: [
  {
   name: 'Cancel'
   func: 'doThis()'
  }
 ]
};
